* 把多个表达式或语句连接成一个复合语句。会返回最后一个表达式计算之后的结果

```js
var a,b,c
return a =1000,
b = a + 2000,
c = b + 3000,
c
```

```js
var a = (a = 1000, a += 2000);
c = (b = (a = 1000, a + 2000), b + 3000)
``` 

* 函数虽然有参数，但是不传参数也是可以调用，只不过各参数的初始值为 undefined
